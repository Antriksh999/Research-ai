# Git Setup Commands for GitHub Deployment

# 1. Initialize Git repository
git init

# 2. Add all files to staging
git add .

# 3. Create initial commit
git commit -m "Initial commit: Research AI Agent for Streamlit deployment"

# 4. Add your GitHub repository as remote (replace with your actual repository URL)
git remote add origin https://github.com/yourusername/research-ai.git

# 5. Push to GitHub
git branch -M main
git push -u origin main

# Alternative if repository already exists:
# git remote set-url origin https://github.com/yourusername/research-ai.git
# git push origin main

# Note: Make sure you've created a repository on GitHub first!
